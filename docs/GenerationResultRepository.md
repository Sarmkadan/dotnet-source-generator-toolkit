# GenerationResultRepository

A repository implementation that stores and retrieves `GenerationResult` objects, typically used to persist source generation results for inspection, debugging, or analysis. It provides asynchronous methods for CRUD operations and querying by common criteria such as entity name, generator type, or status.

## API

### `GenerationResultRepository`

The repository class responsible for storing and managing `GenerationResult` instances.

### `async Task AddAsync(GenerationResult result)`

Adds a new `GenerationResult` to the repository.

- **Parameters**
  - `result`: The `GenerationResult` instance to be added.
- **Exceptions**
  - Throws `ArgumentNullException` if `result` is `null`.

### `async Task<GenerationResult?> GetByIdAsync(string id)`

Retrieves a `GenerationResult` by its unique identifier.

- **Parameters**
  - `id`: The unique identifier of the `GenerationResult` to retrieve.
- **Return Value**
  - Returns the `GenerationResult` if found; otherwise, `null`.
- **Exceptions**
  - Throws `ArgumentNullException` if `id` is `null` or empty.

### `async Task<IEnumerable<GenerationResult>> GetByEntityAsync(string entityName)`

Retrieves all `GenerationResult` instances associated with a specific entity name.

- **Parameters**
  - `entityName`: The name of the entity to filter by.
- **Return Value**
  - Returns an enumerable of matching `GenerationResult` objects.
- **Exceptions**
  - Throws `ArgumentNullException` if `entityName` is `null`.

### `async Task<IEnumerable<GenerationResult>> GetByGeneratorAsync(string generatorName)`

Retrieves all `GenerationResult` instances generated by a specific generator.

- **Parameters**
  - `generatorName`: The name of the generator to filter by.
- **Return Value**
  - Returns an enumerable of matching `GenerationResult` objects.
- **Exceptions**
  - Throws `ArgumentNullException` if `generatorName` is `null`.

### `async Task<IEnumerable<GenerationResult>> GetByStatusAsync(GenerationStatus status)`

Retrieves all `GenerationResult` instances matching a specific status.

- **Parameters**
  - `status`: The `GenerationStatus` to filter by.
- **Return Value**
  - Returns an enumerable of matching `GenerationResult` objects.

### `async Task<IEnumerable<GenerationResult>> GetAllAsync()`

Retrieves all `GenerationResult` instances stored in the repository.

- **Return Value**
  - Returns an enumerable of all `GenerationResult` objects.

### `async Task DeleteAsync(string id)`

Removes a `GenerationResult` from the repository by its unique identifier.

- **Parameters**
  - `id`: The unique identifier of the `GenerationResult` to remove.
- **Exceptions**
  - Throws `ArgumentNullException` if `id` is `null` or empty.

### `async Task ClearAsync()`

Removes all `GenerationResult` instances from the repository.
