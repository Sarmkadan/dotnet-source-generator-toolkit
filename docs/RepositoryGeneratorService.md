# RepositoryGeneratorService

A service class responsible for generating repository pattern implementations for data access in .NET applications. It provides asynchronous methods to create, verify existence of, delete, and count repository artifacts, supporting both individual and bulk operations.

## API

### `RepositoryGeneratorService`

The primary service class that orchestrates repository generation. It encapsulates the logic for transforming source definitions into repository implementations.

### `public async Task<GenerationResult> GenerateRepositoryAsync(...)`

Generates a single repository implementation based on the provided source definition. The method returns a `GenerationResult` indicating success or failure of the operation, including any generated artifacts or error messages.

**Parameters:**
- `source`: The definition or configuration used to generate the repository.

**Return Value:**
- A `Task<GenerationResult>` that completes with a `GenerationResult` object containing the outcome of the generation process.

**Exceptions:**
- Throws `ArgumentNullException` if `source` is `null`.
- Throws `InvalidOperationException` if the generation process encounters an unrecoverable error.

---

### `public async Task<IEnumerable<GenerationResult>> GenerateAllRepositoriesAsync(...)`

Generates repository implementations for all provided source definitions. The method returns an enumerable of `GenerationResult` objects, each corresponding to the outcome of a single repository generation.

**Parameters:**
- `sources`: A collection of definitions or configurations used to generate the repositories.

**Return Value:**
- A `Task<IEnumerable<GenerationResult>>` that completes with a sequence of `GenerationResult` objects, one for each input source.

**Exceptions:**
- Throws `ArgumentNullException` if `sources` is `null`.
- Throws `InvalidOperationException` if the generation process encounters an unrecoverable error for any source.

### `public async Task<bool> ExistsAsync(...)`

Checks whether a repository implementation already exists for the given source definition. Returns `true` if the repository exists, otherwise `false`.

### `public async Task<bool> DeleteAsync(...)`

Deletes an existing repository implementation for the given source definition. Returns `true` if the deletion was successful, otherwise `false`.

**Parameters:**
- `source`: The definition or configuration corresponding to the repository to delete.

**Return Value:**
- A `Task<bool>` that completes with `true` if the repository was successfully deleted, otherwise `false`.

**Exceptions:**
- Throws `ArgumentNullException` if `source` is `null`.
- Throws `InvalidOperationException` if the deletion process encounters an unrecoverable error.

---
### `public async Task<int> CountAsync(...)`

Counts the number of repository implementations that exist for the provided collection of source definitions. Returns the total count of existing repositories.

**Parameters:**
- `sources`: A collection of definitions or configurations to check for existing repositories.

**Return Value:**
- A `Task<int>` that completes with the total number of repositories that exist for the given sources.

**Exceptions:**
- Throws `ArgumentNullException` if `sources` is `null`.
