# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of `dotnet-source-generator-toolkit` seriously. If you discover a security vulnerability, please do not report it through public GitHub issues.

Instead, please report it using one of the following methods:

1. **GitHub Private Advisory:** Use GitHub's built-in private vulnerability reporting feature in this repository.
2. **Email:** Contact us directly at rutova2@gmail.com.

### Response Time

You can expect an acknowledgment of your report within **48 hours**. We will work with you to understand the issue, provide a timeline for a fix, and coordinate the public disclosure.
